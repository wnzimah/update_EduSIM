# Personal Tracker (Schedule + Duit + Profit Bisnes)

Website ini berdiri sendiri dan tidak berkaitan dengan modul LMS.

## Cara guna

1. Buka fail `index.html` dalam browser.
2. Tambah data pada:
   - `Schedule`
   - `Duit Masuk / Keluar`
   - `Business: Kos & Untung`

Semua data disimpan automatik dalam `localStorage` browser anda.

## Ciri

- Rekod tugasan ikut tarikh + status siap.
- Paparan jumlah tugasan untuk esok.
- Rekod transaksi duit masuk/keluar + baki semasa.
- Rekod kos pembelian dan jualan bisnes.
- Kiraan untung bersih automatik.
